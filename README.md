## ■ソフト概要■  
ディスプレイをスタンバイモードに移行するソフトです。  
電源プランのディスプレイの電源を切る時と同じ動作を実行します。  
席を外す際や、PCを起動させたままでディスプレイをスタンバイモードに移行させたい場合などにお使いください。
